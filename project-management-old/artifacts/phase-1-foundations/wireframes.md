## Accessibility Requirements

All screens must meet WCAG 2.1 AA standards:

- **Color Contrast**: Minimum 4.5:1 for normal text, 3:1 for large text
- **Touch Targets**: Minimum 48x48dp for all interactive elements
- **Screen Reader**: Semantic labels on all inputs and buttons
- **Keyboard Navigation**: All interactive elements accessible via keyboard
- **Focus Indicators**: Clear 2dp outline on focused elements
- **Text Scaling**: Support up to 200% text scaling

## Responsive Design Guidelines

### Mobile (Phone)
- Screen padding: 16dp
- Card padding: 16dp
- Grid columns: 1-2 columns
- Bottom navigation: 56dp height

### Tablet
- Screen padding: 24dp
- Card padding: 20dp
- Grid columns: 2-3 columns
- Bottom navigation: 64dp height
